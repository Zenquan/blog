## 聊聊数组的去重问题

> 前端的胖友经常有个需求就是要对数组进行去重。

辣么，怎么去重呢？

ES5实现

```js
var testArr = [1, 1, 2, 2, 3, 3, 4, 5, 5];
function unque(arr) {
    // 判断有没有这个元素
    for(var i=0;i<arr.length;i++) {
        if(arr.indexOf(i)!=-1) {
            arr.splice(i, 1)
        }
    }
    return arr;
}
console.log(unque(testArr));
```

ES6实现

```js
var testArr = [1, 1, 2, 2, 3, 3, 4, 5, 5];
function unque2 (arr) {
	return Array.from(new Set(arr));
}
console.log(unque2(testArr));
```
